# アートシンキング 授業 発表サイト

このリポジトリは、学校のアートシンキングの授業で発表するために制作したサイトの配布用フォルダです。

## 概要

- 制作目的: 授業での発表用に、自分の制作物（成果物）をまとめて見せるための静的ウェブページです。
- 制作期間: 学期内（授業での課題として制作）

## 制作の流れと役割

- HTML: 全て手書きで作成しました。
- CSS: 自分でスタイルを手書きで作成・調整しています。
- AI の活用:
  - サイトのテンプレート作成を AI に依頼してベースを作成してもらいました。
  - その後、AI が生成したテンプレートを元に、見た目（CSS）や一部の構造を自分で学びながら手直ししています。
  - `app.html`（発表用の一部ページ）は AI にベースを書いてもらい、ウィンドウ表現などを自分で勉強して書き換えた箇所があります。

要するに、AI は「補助（テンプレート作成の支援）」として使用し、最終的な実装・調整は自分で行っています。

## ファイル構成（主要なもの）

- `index.html` — トップページ（手書き）
- `app.html` — AI がベースを生成したページ（自分で一部修正あり）
- `style.css` — 全体のスタイル（自作）
- `moc-images/` — モックやデモ画像を格納するフォルダ

（提出用フォルダ内のファイルをそのまま配布しています。）

## 表示方法（ローカルで確認する手順）

1. このフォルダをローカルにダウンロードまたはクローンします。
2. ブラウザで `index.html` または `app.html` を開いて表示を確認してください。
   - 例: ファイルをダブルクリックするか、ブラウザの「ファイルを開く」で該当ファイルを選択します。

（ローカルにサーバを立てる必要はありませんが、念のためブラウザのセキュリティや iframe の挙動によっては一部表示が制限される場合があります。）

## 補足・注意点

- 表示やスタイルは自分が学びながら調整した結果です。細かなレイアウトやブラウザ差異がある可能性があります。
- iframe 内のコンテンツ（もし使用している場合）は、埋め込み先の HTML のスタイルに依存します。スクロールバーの表示非表示など細かい制御が必要な場合は、埋め込み先のファイルも編集してください。
